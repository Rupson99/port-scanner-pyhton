\# Python Port Scanner



\## Description

This is a basic port scanner built using Python to identify open ports on a target system. It was created to understand networking concepts and socket programming.



\## Features

\- Scans ports in a user-defined range

\- Identifies open ports

\- Displays service names

\- Fast scanning using optimized timeout

\- Clean and readable output format



\## How to Run

1\. Make sure Python is installed

2\. Run the program:

&#x20;  python port\_scanner.py

3\. Enter target IP (e.g., 127.0.0.1)

4\. Enter start and end port (e.g., 1 to 1024)
